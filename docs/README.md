# Dossier de conception

Ce dossier contient toute la conception **réalisée AVANT le développement**
(exigence du sujet — des points sont retirés sinon).

## Structure

```
docs/
├── uml/        # Diagrammes UML
│   ├── cas-utilisation/   # min. 22 use cases, acteurs, extend/include/généralisation
│   ├── classes/           # min. 7 classes métier
│   └── sequences/         # min. 7 scénarios + diag d'objets pré/post pour chaque cas
└── uiux/       # Zoning, wireframes, maquettes (Figma/Miro)
```

## Rappel des minimums imposés

- Cas d'utilisation : **22 use cases** minimum
- Classes : **7 classes métier** minimum
- Séquences : **7 scénarios** minimum (avec loop, alt, ref)
- Chaque cas d'utilisation illustré par : objets pré-séquence → séquence → objets post-séquence
- Cohérence vérifiée entre cas d'utilisation et séquences
- Chaque diagramme **commenté et justifié**

## Outils suggérés

- UML : PlantUML, draw.io, ou Mermaid
- UI/UX : Figma, Miro
